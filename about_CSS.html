CSS 속성을 스스로 알아내는 방법

태그 선택자(태그명) < 클래스 선택자(.클래스명) < id(#id명) 선택자 순으로 우선순위가 높음. id는 한 페이지에서 중복되면 안됨. 유일무이해야 함. 당연히 태그를 가리키는 선택자보다는 id를 가리키는 선택자가 더 구체적임. 그래서 우선순위가 높다고 할 수 있다. 효과와 선택자에 대한 이야기를 끝냈다.
