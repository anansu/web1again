Javascript란 무엇인가?

사용자와 상호작용하는 웹페이지를 만들고 싶다는 욕망. HTML은 정적임. 수업의 목적은 주간/야간 모드 버튼을 추가하는 것.

HTML-Javascript의 만남 1 - Script 태그

위의 줄은 자바스크립트를 통해 hello world와 1+1을 그려낸 것임. 핵심은 script 태그를 통해 자바스크립트를 HTML내에서 실행한다는 것임. 요런 on~들을 전부 "event"들이라고 한다. 이걸 써서 사용자와 상호작용 하는 것임.

HTML-Javascript의 만남 2 - event

이렇게 이벤트가 일어났을 때 자바스크립트를 실행시키는 것이 onclick임. 중요한 이벤트로는 "클릭" 혹은 "내용이 변했을 때(on~)"

HTML-Javascript의 만남 3 - console

파일을 만들지 않더라도 아주 가볍게 코드를 실행해야하는 상황들이 있음. 그 경우 개발자도구의 console을 활용시 편하다. 대표적으로 텍스트 세는 .length 등이 있겠지. 혹은 엄청 많은 사람들 중에 추첨해야하는 경우! 미리 작성해둔 코드들을 콘솔에서 활용할 수도 있다는 것. 미리미리 작성해두면 좋다!
